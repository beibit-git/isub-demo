# ИСУБ — демонстрационный интерфейс

Информационная система учёта боеприпасов (ИСУБ) — демо обновлённого интерфейса
с тестовыми данными. Один самодостаточный HTML-файл.

**Открыть:** https://beibit-git.github.io/isub-demo/

- Вход по ролям (тестовые учётные записи на экране входа)
- Разделы: Запасы, Инвентаризация, Планирование, Операции, Интеграция·Сверка, Отчёты, Справочники
- Рабочие фильтры, жизненные циклы документов, переключение RU/KZ и светлой/тёмной темы

> Данные тестовые. Наименования и номера частей — демонстрационные.
